Projects

  • Best Crop Prediction based on Environmental conditions and Economic Factors - The project brings a more scientific approach to crop selection using techniques of IOT and Data Science. The project consists of 3 modules - Hardware kit(to gather real time data), Agri-Analyzer(crop prediction algorithm) and pricing-planner(real time grain pricing from NCDEX to extrapolate the profits after the duration of crop being grown). Using the hardware kit we gathered all the real time data such as temperature, humidity, sunlight, soil-type etc. Once the data is captured for continious 7 days from the kit, the data is normalized into the Agri-Analyzer where using K-means clustering algorithm, we computed the best suitable 5 crops for the farmer using the multi-relational database we gathered for all kinds of crops. The pricing for each of these 5 grains for last 3 years was extracted using NCDEX was captured real-time and using regression analysis we predicted the pricing for each crop for the duration it took to grow. In such a way, we suggested the farmer the best crop which would be environmentally fit and most profitable.

The python script here acts as an external trigger where just by having the server & application parameters and needed access, we can trigger the task externally with no time-dependency on the Qliksense server end.
